# GhostRP Launcher
> Oficjalny launcher do gry **GhostRolePlay**.

---

## Opis
GhostRP Launcher umożliwia łatwe uruchamianie gry **GhostRolePlay** bez konieczności ręcznej konfiguracji. Dzięki launcherowi możesz szybko zacząć rozgrywkę i zawsze mieć aktualną wersję gry.

---

## Instrukcja użytkowania
1. Wejdź na oficjalną stronę GhostRP.  
2. Pobierz najnowszą wersję launchera.  
3. Uruchom pobrany plik:  
   ```text
   GhostRolePlay.exe
